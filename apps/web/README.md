This is the `web` app inside the EvoPalantir pnpm monorepo.

## Getting Started

Run the development server from the repo root:

```bash
pnpm --filter web dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page in `apps/web/src/app/page.tsx`. The page auto-updates as you edit the file.

## Common Commands

```bash
pnpm --filter web dev
pnpm --filter web build
pnpm --filter web start
pnpm --filter web typecheck
```
